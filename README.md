# Developer Portfolio

This project is my personal portfolio website, designed to present my work, technical skills, and development approach.

It highlights applications focused on:
- Dynamic UI rendering
- Structured state management
- Event-driven architecture
- Workflow automation tools

The portfolio itself is built using semantic HTML, responsive CSS, and modular JavaScript to reflect clean and maintainable front-end practices.

## Tech Stack
- HTML
- CSS (Flexbox)
- Vanilla JavaScript

The objective is to demonstrate practical problem-solving and structured code organization rather than reliance on frameworks.
